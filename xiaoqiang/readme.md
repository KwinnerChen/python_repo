## 队列模块

1.rabbitmq：

+ 统一实现get, put接口；
+ 引入全局锁，rabbitmq的链接对象是非线程安全的；
+ 全局链接对象单实例，每个线程一个channel对象；
+ 通过工厂函数返回队列实例，类似存储实例，实例化参数需要自定义队列名称；
+ 配置通过config定义：两个参数QUEUE_MODUL, QUEUE_CONFIG;

2.python队列服务

+ 通过python的进程队列服务实现；
+ 统一get， put接口；
+ 通过工厂函数返回队列实例，实例化需要队列名为参数，端口自动分配；
+ 配置通过config定义：参数与rabbitmq相同，配置保持一致；
+ 可使今后自定义模块成为可能；
